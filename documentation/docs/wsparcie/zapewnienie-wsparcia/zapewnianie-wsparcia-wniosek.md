**Uwagi końcowe:**  
Wniosek należy przekazać do działu HR lub pełnomocnika ds. dostępności (osobiście, mailowo lub przez dedykowany system zgłoszeń).  
Czas rozpatrzenia wniosku – do **30 dni roboczych**.
