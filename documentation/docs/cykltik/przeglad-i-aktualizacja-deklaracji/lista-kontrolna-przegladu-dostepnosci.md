## Sekcja B. Kryteria dostępności – skrócona lista WCAG 2.1 AA

| Nr| Obszar       | Kryterium                    | Ocena (Tak/Nie/Nie dotyczy) | Uwagi |
|----|----------------------|---------------------| --------- | ----- |
| 1  | Alternatywy tekstowe| Wszystkie obrazy i grafiki mają poprawne opisy alternatywne (`alt`). |       |       |
| 2  | Multimedia          | Materiały wideo mają napisy, transkrypcje i audiodeskrypcję. |       |       |
| 3  | Struktura nagłówków | Strony zawierają logiczne nagłówki H1–H6 w kolejności hierarchicznej. |       |       |
| 4  | Linki i przyciski   | Linki i przyciski są opisane jasno i zrozumiale. |       |       |
| 5  | Formularze          | Wszystkie pola formularzy mają etykiety (`label`) i komunikaty błędów.  |       |       |
| 6  | Kontrast            | Kontrast między tekstem a tłem wynosi min. 4.5:1. |       |       |
| 7  | Nawigacja klawiaturą| Wszystkie elementy są dostępne z klawiatury (bez myszy). |       |       |
| 8  | Fokus               | Widoczny znacznik fokusu na aktywnych elementach.  |       |       |
| 9  | Responsywność       | Treść dostosowuje się do rozdzielczości ekranu i urządzenia.  |       |       |
| 10 | Język strony        | Język dokumentu ustawiony (`lang="pl"`). |       |       |
| 11 | Dostępność dokumentów| Załączone pliki PDF i DOCX są dostępne (tagi, struktura, alternatywy). |       |       |
| 12 | Kompatybilność technologiczna| Strona działa poprawnie w czytnikach ekranu (NVDA, JAWS). |       |       |
| 13 | Link „Przejdź do treści”| Na początku strony znajduje się link do głównej zawartości.  |       |       |
| 14 | Nagłówek i stopka   | Stałe elementy są spójne i nie zawierają błędów semantycznych. |       |       |
| 15 | Czasowe elementy    | Użytkownik może zatrzymać automatyczne zmiany (karuzele, slajdy). |       |       |
| 16 | Kolor               | Kolor nie jest jedynym nośnikiem informacji.  |       |       |
| 17 | Treści dynamiczne   | Elementy interaktywne mają odpowiednie atrybuty ARIA.  |       |       |
| 18 | Dostępność na urządzeniach mobilnych | Funkcjonalności zachowane na smartfonach i tabletach.  |       |       |
| 19 | Wersja tekstowa / uproszczona| Istnieje możliwość wyświetlenia wersji uproszczonej. |       |       |
| 20 | Deklaracja dostępności | Deklaracja jest aktualna, poprawnie opublikowana i zawiera wszystkie wymagane informacje.  |       |       |
